# CycleTrack

Ein privater, kostenloser Zyklus-Tracker als Progressive Web App.

## Features

- **Zyklustracking**: Periode, Temperatur, Zervixschleim
- **Fruchtbarkeitsberechnung**: Automatische Berechnung fruchtbarer Tage
- **Temperaturkurve**: Diagramm mit Coverline
- **Symptome & Stimmung**: Krämpfe, Brustspannen, Müdigkeit etc.
- **Datenschutz**: Alle Daten bleiben auf dem Gerät (LocalStorage)
- **Offline-fähig**: Funktioniert ohne Internetverbindung
- **Import/Export**: Daten aus Femometer und anderen Apps importieren

## Installation

1. Öffne die App im Browser
2. Klicke auf "Zum Home-Bildschirm hinzufügen"
3. Die App erscheint wie eine native App

## Daten importieren

Die App unterstützt:
- Femometer CSV Export
- Eigene CSV-Dateien
- JSON Backup

## Datenschutz

- Keine Registrierung nötig
- Keine Daten werden an Server gesendet
- Alles bleibt lokal auf deinem Gerät

## Backup

Erstelle regelmäßig Backups über die Export-Funktion!
